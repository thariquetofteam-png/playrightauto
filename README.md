# Playwright Python Automation Framework

## Overview
This project is a Playwright-based automation framework built using Python and pytest.
It follows industry-standard practices such as Page Object Model, virtual environments,
and clean Git version control.

## Prerequisites
- Python 3.x
- Node.js
- Git
- VS Code
